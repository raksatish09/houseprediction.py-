# 🏡 House Price Prediction using Machine Learning

This project predicts **house prices** using **Linear Regression** based on features like:
- **GrLivArea** (Living Area in square feet)
- **BedroomAbvGr** (Number of Bedrooms)

🔹 **Dataset Used:** [Kaggle House Prices Dataset](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data)  

---

## 🚀 Features
✅ Uses **Google Colab** for execution  
✅ Reads **real-world house data** from Kaggle  
✅ Splits data into **training and testing sets**  
✅ Trains a **Linear Regression model**  
✅ Evaluates with **MAE & RMSE** metrics  
✅ **Visualizes** actual vs. predicted prices  
✅ **Predicts price** for new house inputs  

---

---

## 🎯 Model Performance
📊 **Evaluation Metrics:**  
- **Mean Absolute Error (MAE)**: Measures the average error in predictions.  
- **Root Mean Squared Error (RMSE)**: Measures how much predictions deviate from actual prices.  

📉 **Lower values = Better Model!**
